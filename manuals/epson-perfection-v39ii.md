# Epson Perfection V39II

## Driver

Go to [this link](https://download-center.epson.com/softwares/?device_id=Perfection+V39II&region=FR&language=en) to download the latest driver for your operating system.

### Linux

You can use the [Epson Scan 2](https://download-center.epson.com/f/module/30ef581b-81ba-4f6e-ab0f-6c74117e766b/epsonscan2-bundle-6.7.87.0.x86_64.deb.tar.gz) package.

Extract the archive that contains a weird .deb folder 😤 then install the software with the following command :

```bash
~/Downloads/epsonscan2-bundle-6.7.87.0.x86_64.deb$ ./install.sh
```

Run **Epson Scan 2** to start scanning 😎
